# Physics Simulations: Celestial Mechanics and Fluid Dynamics

This repository contains numerical implementations for two classic problems in computational physics: the gravitational Three-Body Problem and a simplified Navier-Stokes model for thermal convection.

## 1. Three-Body Problem
Numerical simulation of the equations of motion for three bodies under mutual gravitational influence.
* **Objective:** Modeling trajectories and studying the chaotic nature of celestial systems.
* **Key Features:** Integration of Newton's laws of motion and analysis of orbital stability based on initial conditions.
